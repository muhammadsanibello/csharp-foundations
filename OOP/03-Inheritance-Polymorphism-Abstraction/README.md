## Mini Projects

### School Management System
Features:
- Abstract `Person` base class
- Student, Teacher, and AdminStaff inheritance
- Course registration system
- Course assignment system
- Runtime polymorphism using `Person[]`

### Banking Management System V2
Features:
- Abstract `BankAccount` base class
- Savings, Current, and Fixed Deposit accounts
- Transaction history tracking
- Deposit and withdrawal rules
- Overdraft handling
- Minimum balance validation
- Runtime polymorphism using `BankAccount[]`
