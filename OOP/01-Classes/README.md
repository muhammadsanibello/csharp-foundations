# C# OOP - Classes & Constructors

This folder contains my hands-on practice and mini projects while learning **Classes and Constructors** in C#.

Introduction to object-oriented programming fundamentals.

### Topics Covered

- Objects and instances
- Constructors
- Fields and methods
- Static members
- Object initialization
- Partial members
- Finalizers
