# 11 - Events

This module covers the Event programming model in C#. Events provide a safe way for objects to notify other objects when something important happens. They are built on top of delegates and are widely used throughout the .NET ecosystem.

- Using events inside a real project

## Key Takeaways

- Events are a controlled way of exposing delegates.
- Publishers raise events.
- Subscribers react to events.
- EventArgs carries additional information to subscribers.
- Event-driven programming reduces coupling between classes.
