# Async/Await in C#

## Overview

This folder contains my practice exercises for learning asynchronous programming in C#. The goal is to understand how to perform non-blocking operations using `async` and `await`, improve application responsiveness, and execute tasks efficiently.

## Topics Covered

- Creating asynchronous methods
- Using `Task` and `Task<T>`
- `async` and `await`
- Asynchronous file I/O
- Asynchronous web resource access
- Basic parallel programming concepts
- CPU-bound vs I/O-bound operations

## Exercises

### Exercise 01 – Create Asynchronous Task

- Creating and executing asynchronous methods
- Returning `Task` and `Task<T>`
- Using `await`

### Exercise 02 – Asynchronous File I/O

- Reading files asynchronously
- Writing files asynchronously
- Working with asynchronous `FileStream` methods

### Exercise 03 – Asynchronous Web Resource Access

- Sending HTTP requests asynchronously
- Receiving web responses
- Processing downloaded data without blocking the main thread

### Exercise 04 – Parallel Programming

- Basic introduction to parallel execution
- Using `Task.WhenAll()`
- Understanding CPU-bound workloads

## Key Concepts Learned

- Difference between synchronous and asynchronous programming
- Difference between asynchronous programming and parallel programming
- When to use `async/await`
- When to use parallel programming
- Writing responsive and efficient C# applications
