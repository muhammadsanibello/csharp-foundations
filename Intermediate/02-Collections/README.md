# C# Collections

This folder contains my practice exercises, guided learning projects, refactored applications, and real-world simulations focused on mastering C# collections and understanding when to choose the right data structure.

The goal of this module was to understand how collections improve:

- Scalability
- Maintainability
- Data organization
- Performance
- Application design

---

# Topics Covered

- `List<T>`
- `HashSet<T>`
- `Dictionary<TKey, TValue>`
- `Stack<T>`
- `Queue<T>`
