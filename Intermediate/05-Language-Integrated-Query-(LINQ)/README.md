## Topics Covered

- Query Syntax

- Filtering (`Where`)
- Projection (`Select`)
- Sorting (`OrderBy`, `ThenBy`)
- Element Operators (`First`, `Single`, etc.)

- Set Operations (`Distinct`, `Concat`)
- Partitioning (`Take`, `Skip`)
- Aggregation (`Average`)
- Grouping (`GroupBy`)
- Joins
  - Inner Join
  - Group Join
  - Left Outer Join
  - Composite Join
  - Multiple Join
- Immediate vs Deferred Execution
